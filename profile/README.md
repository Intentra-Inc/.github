## Why We Exist

When a human makes a purchase, liability is clear. The human clicked "Buy." If disputed, chargeback rules apply.

When an AI agent makes a purchase on behalf of a human, liability is unclear:

- **The human says:** "I didn't authorize that"
- **The agent platform says:** "Our agent followed instructions"
- **The merchant says:** "We received a valid payment request"
- **The payment processor says:** "We processed what we were told"

**Everyone points fingers. No one has proof.**

Intentra exists to make agent-initiated commerce auditable, defensible, and scalable.

There is no neutral, verifiable, tamper-proof evidence of:
- What authority the human actually granted
- What constraints existed on that authority
- What the agent intended to do
- What the agent actually did
- Who approved it along the way

Without this evidence layer:
- Chargebacks will explode
- Fraud will scale (agents as deniability shields)
- Merchants will refuse agent transactions
- Regulators will intervene
- Lawsuits will have no evidence to resolve them

**Intentra provides the missing evidence layer.**
